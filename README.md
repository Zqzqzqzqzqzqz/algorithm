# Algorithm Atlas

一个零依赖、可直接部署到 GitHub Pages 的经典算法交互式可视化网站。

## 已实现

- 红黑树：自定义插入、删除、随机生成与自动平衡
- B+ 树：自定义阶数、插入、删除、分裂和叶节点链
- DFS：自定义无向图与起点，逐步观察栈、访问顺序与回溯过程
- BFS：自定义无向图与起点，逐步观察队列和逐层扩展过程
- Minimax：自定义深度、分支与叶节点值，完整播放极大极小搜索
- Alpha–Beta：独立剪枝演示，显示 α/β 边界、访问过程与被跳过节点
- Dijkstra：自定义带权图与起点，逐步查看距离松弛
- 排序实验室：快速、归并、冒泡排序，支持自定义数组与动画播放

## 本地运行

这是一个纯静态网站，可以直接打开 `index.html`，或在目录中启动任意静态文件服务。

## 部署

仓库根目录可直接作为 GitHub Pages 的发布目录，无需构建步骤。
